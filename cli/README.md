# cook-cli

灵感来源于 Vue-cli 和 Fis3.

## Useage

- Install

```bash
npm i ck -g
```

- Create Initial Project

```bash
ck init PROJECT_NAME
```

- Create Initial Project with UI

```bash
ck ui
```

## Features
- [ ] `ck init` 由框架模板初始化项目
- [ ] `ck ui` UI可视化
- [ ] `ck dev` 无配置开发模式
- [ ] `ck build` 打包编译
- [ ] `ck deploy` 快速发布
- [ ] 插件支持
- [x] `ck -V` output the version number
- [x] `ck -h` display help for command
